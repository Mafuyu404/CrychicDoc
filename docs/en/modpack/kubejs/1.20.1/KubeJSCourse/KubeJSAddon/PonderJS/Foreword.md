# Foreword

> What is Ponder?

Create immersive scenes for items so players can understand their usage and details more easily.

> Why do some screenshots in this tutorial show purple-black blocks?

Because I was also building a [modpack](https://www.mcmod.cn/modpack/709.html) while writing this tutorial.  
Those purple-black blocks are newly registered `KubeJS` blocks without textures at recording time. In Minecraft, missing textures render as purple-black placeholders.
