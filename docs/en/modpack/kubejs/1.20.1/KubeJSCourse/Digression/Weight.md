# Weight Ratio
Weight is tied to probability: the higher the weight, the higher the chance.

Weight ratio formula: `100 / total weight = per-weight percentage`.

Suppose the weights in one pool are:
1. A weight 3
2. B weight 1
3. C weight 5
4. D weight 2

Calculation: `100 / (3+1+5+2) = 9.09`

So in this pool:
1. A chance is 27.27
2. B chance is 9.09
3. C chance is 45.45
4. D chance is 18.18
