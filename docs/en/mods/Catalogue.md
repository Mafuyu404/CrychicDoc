# Mods

This catalogue groups the maintained mod documentation by theme, so readers can jump straight to the right kind of mod.

## Included Sections

| Section | Focus |
| --- | --- |
| [Adventure Mods](./adventure/Catalogue) | Combat, elite mobs, difficulty scaling, and adventure-oriented gameplay additions. |
| [Armor & Tools](./armor_tools/Catalogue) | Equipment, survival gear, and tool-focused mod documentation. |
| [Cosmetic Mods](./cosmetic/Catalogue) | Visual, emote, and appearance-focused mods that do not mainly change progression. |
| [Custom Content](./custom/Catalogue) | Internally maintained systems, custom projects, and authoring workflows. |
| [Performance](./performance/Catalogue) | Client or server optimization notes, tuning, and compatibility guidance. |
| [Server Mods](./server/Catalogue) | Server-only utilities, administration helpers, and multiplayer support mods. |
| [Storage Mods](./storages/Catalogue) | Storage networks, logistics, terminals, and item-management mods. |
| [Tech Mods](./tech/Catalogue) | Automation, machines, and progression-heavy technical content. |
