# 自定义物品模型

添加一个物品类

```java
public class RubyWand extends Item {
    public RubyWand(){
        super(new Properties().stacksTo(1));
    }
}

```

注册物品

```java
    public static final Supplier<Item> RUBY_WAND = ITEMS.register("ruby_wand", RubyWand::new);


```

添加模型和材质

放到对应的路径下就可以了，和之前一样的。

```
│  ├─examplemod
│  │  ├─blockstates
│  │  ├─models
│  │  │  ├─block
│  │  │  └─item
│  │  └─textures
│  │      ├─block
│  │      └─item

```
