# Random

>[!TIP] 提示
>此处介绍Internal.Random的相关信息。

## 类型定义

- 通过ProbeJS查看其API文档。
- 其实现RandomGenerator与Serializable接口，意味着其具有生成随机数的功能，Serializable只是一个标记其为可序列化的类的接口，其没有定义任何方法，这里无需关注。

## 实例创建

- 通过Utils的getRandom方法获取Random对象，注意该方法始终返回同一个Random对象。

```js [KubeJS]
let random = Utils.getRandom();
```
