# 什么是KubeJS {#Introduction}

**KubeJS**是基于[Rhino](https://github.com/mozilla/rhino)使用JavaScript编写魔改脚本的老牌 魔改 模组。它好上手、魔改支持面广，有着较好的插件生态，通过事件驱动的逻辑它能够轻松实现复杂的数据包才能构建的效果，并能够直接导入原版与模组的类以进行深度开发。

由于其使用的魔改Rhino并非现代化的JS -> Java编译器 ，该模组存在许多限制与遗留问题，且在未来应当也不会被解决。

*[魔改]: 使用配置文件、数据包、脚本甚至开发新的模组来对原版与模组的行为进行定制与修改。
